The org.jasig.cas.event.advice package contains the AOP Advices that are used to advise some of the CORE CAS classes in order to implement the cross-cutting concern of publishing events such as AuthenticationEvent, PageRequestHttpRequestEvent and TicketEvent.

The AuthenticationHandlerAfterReturningAdvice advises the AuthenticationHandler interface. The CentralAuthenticationServiceAfterReturningAdvice advises the CentralAuthenticationService interface. The PageRequestHandlerInterceptorAdaptor is a web tier concern that publishes events about page renderings.
