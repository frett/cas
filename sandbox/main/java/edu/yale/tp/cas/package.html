This package contains XMLBeans-generated interfaces that handle the CAS 2.0 XML responses. These interfaces may be replaced by your own XMLBeans-generated interfaces if you need to extend the CAS 2.0 schema. You will need to replace the contents of the following directories with the XMLBeans-generated files:

core/src/main/java/edu/yale/tp/cas
core/src/main/java/edu/yale/tp/cas/impl
core/src/main/java/schema
webapp/WEB-INF/classes/schema
